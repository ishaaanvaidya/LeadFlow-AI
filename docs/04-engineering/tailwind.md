# Tailwind Standards

Tailwind CSS v4 is the styling system. Styling should be consistent, readable, and tied to the design system.

## Rules
- Use Tailwind utilities and design tokens.
- Avoid inline styles.
- Avoid CSS Modules and styled-components.
- Extract repeated UI patterns into components.
- Keep class strings readable by grouping layout, spacing, typography, color, and state.

## Theme
Use neutral grayscale foundations with blue accent, green success, amber warning, and red danger. Support dark mode.
