# AI Insights

Insights turn lead context into concise next-step suggestions.

## Insight Types
- Suggested next action.
- Risk or missing information.
- Timing suggestion.
- Personalization angle for outreach.

## Rules
- Limit MVP insights to 2-3 items.
- Keep each insight actionable.
- Explain which signal led to the suggestion where possible.
- Avoid confident claims not grounded in provided data.
