# Project Summary

LeadFlow AI is an AI-native CRM MVP for a portfolio/demo SaaS. It manages leads, pipeline stages, activity history, dashboard metrics, and AI-assisted email drafting, scoring, and insights.

The project is currently in Engineering OS setup. Feature implementation has not started beyond the default Next.js scaffold.

## Current Architecture
- Next.js `16.2.9` App Router.
- React `19.2.4`.
- TypeScript.
- Tailwind v4.
- Planned Prisma, Better Auth, Zod, AI provider adapter, and CRM feature modules.
