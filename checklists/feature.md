# Feature Checklist

## Checklist
- Relevant docs read.
- Scope matches PRD or approved task.
- Domain impact understood.
- Validation defined.
- Auth and ownership handled.
- UI states implemented.
- Tests or manual checks completed.
- Documentation updated if architecture changed.
