# UI Checklist

## Checklist
- Responsive at 375px and desktop.
- Dark mode readable.
- Keyboard navigation works.
- Focus states visible.
- Empty, loading, error, and success states exist.
- Text does not overflow controls.
- Icons have labels or tooltips where needed.
- UI supports the workflow without decorative clutter.
