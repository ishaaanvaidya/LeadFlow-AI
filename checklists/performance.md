# Performance Checklist

## Checklist
- Server Components used where appropriate.
- Client boundary is small.
- Lists are paginated or bounded.
- Queries use likely indexes.
- Loading states exist.
- Heavy libraries are justified.
- AI calls do not block core CRM rendering.
