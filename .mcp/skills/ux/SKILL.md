# UX Skill

## Purpose
Keep workflows clear, fast, accessible, and aligned to CRM user needs.

## Required Reading
- `docs/01-product/personas.md`
- `docs/01-product/workflows.md`
- `docs/05-design/design-system.md`
- `docs/05-design/accessibility.md`

## Rules
- Optimize repeated sales workflows.
- Provide clear empty/loading/error states.
- Keep AI suggestions editable.
- Avoid decorative friction.

## Forbidden Shortcuts
- Marketing hero pages for app workflows.
- Hidden controls without labels.
- Toast-only critical feedback.

## Handoff Format
List user flow, states, accessibility concerns, and tradeoffs.

## Review Checklist
- Clear next action.
- Mobile usable.
- Keyboard usable.
- Text fits.

## Acceptance Criteria
The user knows what happened and what to do next.
